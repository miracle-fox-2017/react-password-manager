# React Password Manager
An application managing password using react, redux, jsx, and firebase as backend

├── src

│   ├── App.css

│   └── store

```

## Use this app?
```
$ git clone git@github.com:fariswd/react-password-manager.git
$ cd react-password-manager
$ npm install
$ npm start
``
